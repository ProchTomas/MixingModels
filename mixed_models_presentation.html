Introduction to the Problem

Physical Machine Settings

  • Materials science experiments involve expensive, time-consuming physical runs.
  • The Problem: Very low amounts of data relative to the configuration space.
  • Standard regression models overfit or fail to capture LOCALIZED variances.

Problem Definition: g, z, y

Input Features

Settings Vector g: Categorical

e.g., [frequency, deposition time, substrate, ...]

Regressors z: Numerical

Physical continuous inputs (initial and working pressure).

Output Target

Response y: numerical

Observed physical outcome of the setting (thickness).

y ^ ∼ p ( y | z, g, Θ )

Approach: Mixed Sub-Models

Instead of one complex model, we build a Mixture of Experts:

Sub-Model Unit

  • For each category in each entry of g: g_ij.
  • Simple 1D Linear Adaptive Model.
  • Initiated with non-informative priors.
  • Updates via recursive Bayesian estimation.

Adaptive Learning

  • Employs Optimal Forgetting.
  • Adapts to system shifts automatically.
  • Calculates localized sufficient statistics.

The Mixing Logic

Combining sub-models into a unified forecast by minimizing cross-entropy:

F ( ϕ i ) = KLD ( ϕ i ∥ ϕ i , 0 ) − ∫ ∏ j = 1 l g p ( y i | z i , g i j , θ i j ) d y

Optimization yields Optimal Mixing Weights (ϕ).

  • Compresses complex posterior logic into a single predictive distribution.
y ^ i = ∑ j = 1 l g ϕ j y ^ i j

The Goal: Recommendation

Inverse Optimization

Search for the optimal g.

  • Given a target response y*.
  • Exhaustive search over categorical combinations.
  • Recommends settings to achieve the physical goal.

Validation: Leave-One-Out (LOO)

Methodology

  1. Hold out one observation y, z, g.
  2. Train the mixture on all remaining N-1 points.
  3. Predict the held-out point using mixed modeling.
  4. Repeat for all N observations.

Benchmarked against two standard references:

  • Mean Baseline: Average of N-1 observations.
  • Simple OLS: Ordinary Least Squares on continuous z only.
